# FisioTareaLuis — Registro de Cita de Fisioterapia

## Descripción
Formulario web para el registro de citas en un sistema fisioterapéutico.
Permite registrar datos del paciente, historial de citas, evaluación
de la consulta actual y plan de tratamiento.

## Contenido del repositorio
- `index.html` → Formulario principal con semántica HTML5
- `style.css` → Estilos visuales del formulario
- `wireframe.pdf` → Boceto del diseño del formulario

## Etiquetas semánticas utilizadas
- `<main>` → Contenedor principal del contenido
- `<header>` → Encabezado con el nombre del sistema
- `<section>` → Agrupa cada bloque del formulario
- `<form>` → Formulario de registro
- `<table>` → Historial de citas previas
- `<label>` → Etiquetas accesibles para cada campo
- `<textarea>` → Campos de texto largo
- `<select>` → Listas desplegables

## Tipos de entrada utilizados
- `type="text"` → Nombre del paciente
- `type="number"` → Cédula y edad
- `type="date"` → Fecha de cita previa
